## Core Concepts

<CardGrid>
  <Card title="Transpose" icon="random">
    Generic 2-level attribute set transposition: `{ a.b = v }` → `{ b.a = v }`. Parameterized by an `emit` hook.
    <LinkButton href="/concepts/transpose" variant="minimal" icon="right-arrow">Learn More</LinkButton>
  </Card>
  <Card title="Aspects & Resolution" icon="puzzle">
    Aspect submodules with `includes` (dependency graph) and per-class configs. Recursive resolution collects all transitive class-specific modules.
    <LinkButton href="/concepts/aspects" variant="minimal" icon="right-arrow">Learn More</LinkButton>
  </Card>
  <Card title="Providers & Fixpoint" icon="setting">
    Nested sub-aspects via `provides` / `_`. Fixpoint semantics let providers reference siblings and top-level aspects.
    <LinkButton href="/concepts/providers" variant="minimal" icon="right-arrow">Learn More</LinkButton>
  </Card>
</CardGrid>

  </Card>
  <Card title="Without Flakes" icon="open-book">

  </Card>
  <Card title="Cross-Aspect Dependencies" icon="list-format">

  </Card>
  <Card title="Parametric Aspects" icon="seti:config">

  <Card title="__functor Override" icon="pencil">

    <LinkButton href="/guides/functor" variant="minimal" icon="right-arrow">Learn More</LinkButton>
  </Card>
  <Card title="Forward Across Classes" icon="right-arrow">
